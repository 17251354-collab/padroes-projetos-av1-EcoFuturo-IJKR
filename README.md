# padroes-projetos-av1-EcoFuturo-IJKR
EcoFuturo Esuda - Plataforma gamificada que incentiva a mobilidade sustentável através de recompensas. Usuários registram atividades ecológicas (bicicleta, caminhada, transporte público, carro elétrico), acumulam moedas e trocam por benefícios com parceiros locais. Projeto para disciplina de Padrões de Projeto
